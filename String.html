- (Object) spread_line(tabstops = [], search = /,/, position_add = nil, start = 0) Originally defined in module Perception::StringSpreadI

Erzwingt ein bestimmtes Zeichen nur an Tabstop-Positionen Mit position_add= 0 werden die Spaces vor dem gefundenen Zeichen eingefügt. Standard ist position_add= search.size, damit werden die Spaces hinter den gefundenen Zeichen eingefügt. Mit PPP_ANTI_NEWLINE werden Zeilen markiert, bei denen anschließend noch das n entfernt werden kann.
